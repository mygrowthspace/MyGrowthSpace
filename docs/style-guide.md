# Style Guide: My Growth Space

## 1. Visual Identity
The brand identity is built on clarity, progress, and scientific reliability. The interface should feel "airy," professional, yet encouraging.

## 2. Color Palette

### Primary (Action & Brand)
- **Indigo-600**: `#4f46e5` - Primary actions, active navigation icons.
- **Indigo-500**: `#6366f1` - Secondary highlights, branding accents.
- **Indigo-50**: `#eef2ff` - Subtle backgrounds for cards or selections.

### Typography (Contrast & Readability)
- **Slate-900**: `#0f172a` - Primary Headings, input text, active body text.
- **Slate-700**: `#334155` - Secondary body text, subheadings.
- **Slate-500**: `#64748b` - Tertiary info, labels, placeholders.
- **Slate-400**: `#94a3b8` - Disabled states, icons in inactive states.

### Semantic Colors
- **Emerald-600**: Success, completed habits, positive growth.
- **Amber-500**: Warning, tips, premium features.
- **Red-500**: Destructive actions (Delete, Reset).
- **Slate-50**: Backgrounds for the entire app.

## 3. Typography
- **Primary Font Family**: 'Inter', sans-serif.
- **Weights**:
  - 400 (Regular): Body text.
  - 500 (Medium): UI Labels, navigation.
  - 600 (Semi-bold): Subtitles, card titles.
  - 700 (Bold): Main headings.
  - 800 (Extra-bold): Large display onboarding text.

## 4. Components & Elevation

### Cards
- **Background**: White (`#ffffff`).
- **Border**: 1px solid `Slate-100` or `Slate-200`.
- **Radius**: `2xl` (1rem) or `3xl` (1.5rem).
- **Shadow**: `shadow-sm` for normal cards, `shadow-md` for interactive/hovered cards.

### Buttons
- **Primary**: Indigo-600 background, White text.
- **Ghost/Tertiary**: Slate-50 background, Slate-700 text.
- **Destructive**: White background, Red-500 text.

## 5. Accessibility Rules
- **Text on White**: Minimum color weight `Slate-700`. Use `Slate-900` for critical information.
- **Text on Indigo-600**: Must be `White`.
- **Inputs**: Text must be `Slate-900` to ensure users can read their own entries clearly.
